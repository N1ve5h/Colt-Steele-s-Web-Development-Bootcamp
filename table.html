How to Make a Table?

The table element represents tabular data - that is information presented in a two-dimensional table comprised of rows and columns of cells containing data.

The tr element defines a row of cells in table. The row's cells can then be established using a mix of td (data cell) and th (header cell) elements. Here is an example:
